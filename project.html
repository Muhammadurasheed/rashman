Nwaste

Nwaste is a React and Firebase full-stack application designed to enhance the agricultural industry's efficiency, productivity, sustainability, and profitability. Its features include a recommendation system, loan section, product upload for farmers, marketplace, and integration with Google Maps and Stripe. The Firebase backend offers secure authentication and real-time database services, while the application is optimized for a native mobile experience using PWA technology.

Math Sprint

Math Sprint is a JavaScript-based math quiz application featuring randomized questions, a timer, automatic scoring, and review function. The user-friendly interface allows users to test their math skills under pressure, and review their performance after the quiz. The application uses HTML/CSS for styling and third party API for questions bank.

Drawboard

Drawboard is a cutting-edge JavaScript application that leverages the latest canvas technology to provide users with a sophisticated drawing and painting experience. With Drawboard, users can easily create, edit, erase, and save their artwork directly to local storage.
