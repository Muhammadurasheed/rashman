Nwaste

Nwaste is a full stack application built with React and Firebase. It is designed to solve the lack of efficiency, productivity, sustainability, and profitability in the commercial agricultural industry. The features and the technologies used in the application are:
1. Recommendation system through which crops are recommended to farmers based on demand.
2. Loan section where farmers can have access to global agricultural loans and grants.
3. Ability to upload products when signed in as a farmer.
4. Marketplace where farmers are connected to potential customers (buyers) seamlessly.
5. Google Maps was integrated in the frontend of the application to provide location-based services. Google Map was used in the application to compute the distance and the time for the arrival of product.
6. Firebase, the server-side component of the application was used as the backend platform to handle requests from the frontend and process data. It was used in providing authentication and real-time database services which are responsible for the secure and scalable web application.
7. Progressive Web App (PWA) was the technology that allows the web applications to function like native mobile apps, providing users with an immersive and responsive experience.
8. Stripe was integrated into the frontend of the application to facilitate online payments. Stripe facilitates secure credit card transactions and funds transfer between buyers and sellers.

Math Sprint

Math sprint is a math quiz application developed with JavaScript that allows users to take random math questions with a timer and provides an automatic score at the end of the quiz. The application features a user-friendly interface and provides users with the flexibility to review their performance and see the questions they got right and those they got wrong.

Key Features:
✔ Randomized questions: The application provides users with random math questions from a bank of questions, ensuring that each quiz is unique and challenging.
✔ Timer: The quiz is timed, allowing users to test their math skills under pressure.
✔ Automatic scoring: The application provides an automatic score at the end of the quiz, making it easy for users to assess their performance.
✔ Review function: The application allows users to review their performance and see the questions they got right and those they got wrong, enabling them to identify areas for improvement.

Technologies Used:

1. JavaScript: The application was developed using JavaScript, a popular programming language used for building web applications.
2. HTML/CSS: The application uses HTML and CSS for the user interface and styling.
